What is the difference between innerHTML, innerText, and textContent ?
  1. innerHTML Reads or writes HTML content,
  2. innerText Reads or writes only visible text
  3. textContent Reads or writes all text content

What is event bubbling in the DOM?
The concept of event bubbling in JavaScript is that an event starts at the target element and then propagates to its ancestors in the DOM tree.
